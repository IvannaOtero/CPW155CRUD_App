# CPW155CRUD_App
.NET application to demonstrate CRUD (
Create, Retrieve, Update, Delete) operations with 
as SQL Server database using ADO.NET. 

## Getting Started 
- Install .NET 10 
- Recommended IDE: VS 2026
- Clone the repo 
- Install SQL Server developer with dafault settings 
- Run 'TennisDb.sql' againsts localhost 
- Run the app
